**Simple JS Calculator**

**Short Description :**
A sleek, responsive calculator web application built with vanilla JavaScript, HTML5, and CSS3. Features standard arithmetic operations with a modern UI.

**Key Features**

**Basic Arithmetic**: Supports addition, subtraction, multiplication, and division.

**Responsive Design:** Fully functional on mobile, tablet, and desktop screens.

**Clear & Delete:** Includes functionality to clear the entire screen or delete the last digit entered.

**Tech Stack**

**HTML5:** Structured the calculator layout.

**CSS3:** Styled the interface using Flexbox/Grid for a clean look.

**JavaScript (ES6):** Handled the logic, event listeners, and mathematical computations.

**How to Run It**
**Clone the repository:** 

git clone [https://github.com/taha-karim/simple-calculator.git]

Open index.html in your favorite browser.

**Start calculating!**
